## Overview
The Library Management System is a simple project that helps manage library records digitally. It reduces manual work and makes book and user management more organized and efficient.

## Problem Statement
Manual library systems are time-consuming and prone to errors. Managing books, users, and records becomes difficult without a proper digital solution.

## Objectives
- Reduce manual work
- Maintain accurate book records
- Make searching and tracking easier
- Improve overall library efficiency

## Technologies Used

- MySQL

## Features
- Add and view book records
- Manage user details
- Issue and return books
- Simple and user-friendly interface
